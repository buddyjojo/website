Install Instructions

How to make your device even more sus.

1. Download the latest files from our Downloads page for your architecture.

2. Flash the image to your device:

  • For RPi Version: Flash the img.xz file with Raspberry Pi Imager or Balena Etcher. (If it gets stuck at a blinking cursor while booting for more than 1 minute, reboot it and spam the ESC key)

  • x86_64 Version: Import the .ova file into an VM software, or flash the image to your USB stick to run it on bare metal. (Instructions for which can be found in a detailed instructions file.)


3. Boot it up and it's done! Enjoy!
